#### **Social-Engineering-Attack-Lifecycle (Academic Lab)**
Simulated an end-to-end social engineering attack to analyze human-centric vulnerabilities and payload delivery mechanisms.
* **Tasks:** Developed a **C++ payload** using **MSFVenom** shellcode; deployed a phishing campaign via a simulated **Roundcube Webmail** portal; configured a **Metasploit** listener to capture a reverse TCP shell.
* **Focus:** Identifying indicators of compromise (IoC) and analyzing how attackers bypass technical controls through user deception.
